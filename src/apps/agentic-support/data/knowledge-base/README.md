## How to Convert to PDF

### Option 1: Using macOS (Recommended)
1. Open the `.txt` file in TextEdit or any text editor
2. Go to **File > Print**
3. In the print dialog, click **PDF > Save as PDF**
4. Save with a descriptive name (e.g., `printer-troubleshooting.pdf`)

### Option 2: Using Online Converters
1. Visit an online text-to-PDF converter (e.g., ilovepdf.com, smallpdf.com)
2. Upload the `.txt` file
3. Convert to PDF
4. Download the PDF file

### Option 3: Using Command Line (macOS)
```bash
# Install textutil (comes with macOS)
textutil -convert pdf printer-troubleshooting.txt -output printer-troubleshooting.pdf
```

### Option 4: Using Microsoft Word
1. Open the `.txt` file in Microsoft Word
2. Format as needed (optional)
3. Go to **File > Save As**
4. Choose **PDF** as the file format
5. Save the file

## Upload Instructions

1. Navigate to **Agentic Support > Knowledge Base** in the application
2. Create a new category matching the document (e.g., "Printer Troubleshooting")
3. Click **Upload PDF Document**
4. Select the converted PDF file
5. Wait for processing and vectorization to complete
6. Verify the vector count is displayed
